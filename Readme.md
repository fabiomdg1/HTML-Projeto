## Nome do Projeto

Pet's Dog

## Descrição

Trata-se de um site construido utilizando apenas HTML5 e algumas das tags semânticas.

## Objetivo
Utilizar apenas HTML sem o auxilio do CSS afim de fixar os conceitos aprendidos no curso.
